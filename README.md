# ALX Travel App

## Overview
The **ALX Travel App** is a Django-based platform that serves as the foundation for a travel listing service. It includes MySQL integration, REST API setup, and Swagger documentation.

## Features
- Django REST Framework for API support
- MySQL database with environment variable configuration
- Swagger UI for API documentation at `/swagger/`
- CORS headers for cross-origin API access
